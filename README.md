# 📝 Text Processing – Python Fundamentals

This repository contains exercises and examples focused on **text processing in Python**.  
The goal is to practice working with **strings, text manipulation, and basic algorithms** using Python fundamentals.

It is part of my learning journey while studying **Python Fundamentals**.

---

## 📚 Topics Covered

The repository includes exercises and solutions related to:

- Working with **strings**
- **String slicing and indexing**
- **Text parsing**
- **Splitting and joining text**
- **Searching in text**
- **Replacing text**
- **Character manipulation**
- **Basic algorithms for text processing**

---

## 🛠 Technologies

- **Python 3.12**
- **PyCharm**
- **Virtual Environment (venv)**
