# Python Fundamentals – Text Processing

This section focuses on **Text Processing** in Python.

Text processing involves working with strings, modifying text, extracting information, and solving problems related to text manipulation.

---

## Topics Covered

### Lab
The **Lab** tasks introduce the basics of working with text and strings.

They cover:
- String manipulation
- Iterating through characters
- Basic string methods (`split()`, `replace()`, `join()`)
- Simple text transformations

These tasks help build a strong foundation for working with textual data.

---

### Exercise
The **Exercise** section includes more complex problems that require combining string operations with logic.

You will practice:
- Parsing and extracting data from text
- Modifying strings based on conditions
- Working with lists of strings
- Building formatted output

These tasks improve problem-solving skills and prepare you for real-world scenarios involving text processing.

---

### Bonus Exercises
The **Bonus Exercise** tasks are more challenging and focus on deeper understanding.

They involve:
- Complex string manipulations
- Handling edge cases in text
- Combining text processing with other data structures
- Writing clean and efficient solutions
